Class stack_of_object

openssl.stackofobject object stackofx509extension, stackofx509, stackofx509attribute object has same interface. stackofx509 is an important object in lua-openssl, it can be used as a certchain, trusted CA files or unstrust certs. object not support x509, x509extension or x509attribute
